## Youtap assesment test

#### Requirement
 Given the string of "(" or ")". Create a function to validate the input with the balanced bracket.

 ### How to run
 1. Run directly in local
  * Nodejs version v14.17.3
  * run `yarn install`
  * run `yarn run test` 
2. Use docker-compose
  * run `docker-compose up`
